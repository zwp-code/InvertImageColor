# Web-OS 网页操作系统

一款在线的仿Win11网页操作系统（未完善）

开源地址：Gitee (opens new window)

# 技术栈

基于Vue3.0+Element Plus实现的仿win11操作系统

# 功能介绍

整体界面主要仿win11的界面风格

目前实现的功能包括：

1.整体界面的搭建

2.浏览器的实现，采用iframe内联框架实现，包含了搜索，页面标签，窗口缩放，移动，隐藏，页面历史前进后退，刷新等功能

除了内置的浏览器外，还有其他普通的应用，文件管理器尚未完成....

其他应用可以在config里面进行配置，可以配置是否外部打开还是内联框架打开，不过有些网页在内联框架下会出现无法访问的情况

由于这里接入了bmob云后端，所以需要一开始登录账号和密码才能访问(已经废弃)

登录的账号是admin，密码是123456

# 项目启动

# 依赖安装

npm install

# 项目运行

npm run serve

# 项目打包

npm run build
