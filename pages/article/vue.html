# Vue 开发配置



# 一、创建工程

# 1.使用 vue-cli 创建

官方文档 (opens new window)

## 查看@vue/cli版本，确保@vue/cli版本在4.5.0以上
vue --version
## 安装或者升级你的@vue/cli
npm install -g @vue/cli
## 创建
vue create <your-project-name>
## 或者通过图形化界面创建
vue ui

## 启动
cd <project-name>
npm run serve
